# WAVE.js
Wave file convert ( 8/16/24/32 bits, sampling frequency, channel ) and Analysis, Raw data effects ( volume, speed, octave )  
( Waveファイル変換。 8/16/24/32bitの相互変換。サンプリング周波数、チャンネルの変更。解析、生データの操作、ボリューム、再生速度、オクターブの変更 )
